# Snake

Petit Snake web, modulaire et sans dependance runtime.

## Commandes

```sh
npm test
npm start
```

`npm start` sert le dossier sur `http://localhost:4173`.

## Reglages

Le menu `Reglages` propose les vitesses `Lente`, `Normale`, `Rapide` et
`Expert`, des themes de plateau, une bascule de grille et la map des nouvelles
parties. Ces choix sont sauvegardes localement et s'appliquent a `Rejouer`
comme aux prochaines parties. Le bouton `Defaut` revient aux reglages standards.

## Couleur du serpent

Le menu `Reglages` permet aussi de choisir une couleur de serpent lisible sur
les themes disponibles. L'option `Garder pour Rejouer` conserve ce choix; si
elle est desactivee, `Rejouer` revient au vert par defaut.

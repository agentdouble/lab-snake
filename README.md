# Snake

Petit Snake web, modulaire et sans dependance runtime.

## Commandes

```sh
npm test
npm start
```

`npm start` sert le dossier sur `http://localhost:4173`.

## Reglages

Le menu `Reglages` propose les vitesses `Lente`, `Normale`, `Rapide` et
`Expert`, des themes de couleur et une bascule de grille. Ces choix sont
sauvegardes localement et s'appliquent a `Rejouer` comme aux prochaines
parties. Le bouton `Defaut` revient aux reglages standards.
